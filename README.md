# Resolução da Lista de Revisão I - Programação de Soluções Computacionais

Este repositório contém a resolução detalhada da primeira lista de revisão da disciplina de **Programação de Soluções Computacionais** do Centro Universitário UNA. A lista aborda conceitos fundamentais da linguagem Java, incluindo tipos de dados, operadores, arrays, Strings, exceções e lógica de programação.

## 📋 Conteúdo da Revisão

O material original contém 10 questões de múltipla escolha focadas em sintaxe e comportamento da linguagem Java:

1.  **Operações Aritméticas:** Comportamento da divisão inteira e tipos de dados.
2.  **Operadores de Incremento:** Pós-incremento (`x++`) vs Pré-incremento (`++x`).
3.  **Arrays:** Sintaxe de declaração e inicialização.
4.  **Strings:** Concatenação e manipulação básica.
5.  **Método Main:** Assinatura correta para execução de programas.
6.  **Tratamento de Exceções:** Fluxo de erros não tratados.
7.  **Instanciação:** Criação de objetos com a palavra-chave `new`.
8.  **Lógica Booleana:** Avaliação de expressões com operadores lógicos `&&` e `||`.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java
* **Conceitos:** Lógica de Programação, POO básica e Estruturas de Dados.
